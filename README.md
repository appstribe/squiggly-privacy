# Squiggly — Privacy Policy

This repository hosts the public privacy policy for
**Squiggly: Kids Drawing & Coloring App** (Android).

Live page → https://appstribe.github.io/squiggly-privacy/

The app collects **no personal data**, contains no ads, no analytics,
no third-party SDKs, and runs fully offline. See `index.html` for the
full policy.

Maintained by [@appstribe](https://github.com/appstribe).
